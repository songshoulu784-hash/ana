#其分析结果：
#（1）用户分析

明末游戏整体评论2w条左右，其推荐率为53.22%,退款率为2.2%；用户两周内平均游戏时长28h左右，大部分玩家玩到33h左右。
如果按照游戏时长划分玩家类群，则随着核心与深度玩家推荐率高。而根据其他组群、库存数等加权评估划分玩家类群，评论区大部分为轻度玩家，但同时随着玩家硬核程度推荐率逐渐上升。

#（2）评论内容分析

明末游戏的评论，关键词集中在优化、boss等方面，推荐与不推荐玩家也主要集中这几个方面。通过情感分析，能得出大部分玩家评论较为正向讨论游戏内核：关于性能机制、关卡设计等方面，但随着时间流逝，新人流逝率高。
